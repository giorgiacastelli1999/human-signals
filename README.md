# human-signals
AI-powered behavioral analytics platform for emotional pattern tracking and burnout insights.
## ProblemBurnout is invisible until it's too late.

## Solution  Track mood, stress, energy, sleep → AI detects patterns → actionable insights.

## Stack  - Frontend: Streamlit  - Backend: FastAPI  - Database: PostgreSQL (Neon)  - AI: OpenAI GPT-4  - Analytics: Pandas + Plotly  ## Features  - Daily check-ins  - AI sentiment + emotion extraction  - Trend dashboards  - Burnout risk scoring  

## Business Model  Free: basic trackingPremium: advanced analytics, weekly AI summaries
