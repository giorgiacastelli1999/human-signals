# HumanSignals

HumanSignals is an AI-powered behavioral analytics platform that helps users understand how daily habits influence mood, stress, energy, and burnout risk.

## The Problem

Burnout and emotional decline often develop gradually and are difficult to recognize early. Most tracking apps collect data but provide limited insight into what is actually driving changes in wellbeing.

## The Solution

HumanSignals combines behavioral tracking, analytics, and AI-generated insights to help users identify patterns in their emotional health.

Users can track:

* Mood
* Stress
* Energy
* Sleep
* Exercise
* Social interactions
* Music listening
* Meditation
* Time spent outdoors

## Features

* Daily emotional check-ins
* Behavioral analytics dashboard
* Mood, stress and energy trends
* Burnout risk scoring
* Personalized behavioral insights
* AI-generated reflections

## Tech Stack

* Frontend: Streamlit
* Backend: FastAPI
* Database: PostgreSQL (Neon)
* Analytics: Pandas & Plotly
* AI: OpenAI API
* Deployment: Streamlit Cloud & Render

## Architecture

Streamlit → FastAPI → PostgreSQL → Analytics → AI Insights
